## 🔧 Scripts
```bash
# instalar dependências
npm install

# rodar localmente
npm run dev

# rodar testes
npm run test

# rodar testes com cobertura
npm run test:ci
```

## 🧪 CI/CD
Este projeto usa GitHub Actions para:

- Testes automatizados
- Geração de coverage
- Deploy automático para a Vercel (somente na branch main)
